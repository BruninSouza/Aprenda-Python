## Sobre o projeto

O **Aprenda Python** é um projeto de ensino de Python desenvolvido como um site interativo. Criado com a ferramenta Jupyter Book, seu objetivo é democratizar o aprendizado de programação, tornando-o acessível a qualquer pessoa, mesmo sem um ambiente de desenvolvimento local.

## Tecnologias Utilizadas

- **Python**: A linguagem de programação central do curso.

- **Jupyter Book**: Ferramenta que transforma notebooks Jupyter em um site interativo.

- **Google Colab**: Plataforma que permite a execução dos notebooks diretamente no navegador, sem a necessidade de instalação.

## Funcionalidades Principais

- **Aulas Interativas**: O conteúdo é apresentado em notebooks que podem ser executados diretamente no site.

- **Acessibilidade**: O site tem link direto com o Google Colab. Isso permite que mesmo pessoas sem computadores consigam acessar o site e aprender a programar em Python.

- **Navegação Clara**: O conteúdo é organizado em capítulos, facilitando o acompanhamento da jornada de aprendizado.
